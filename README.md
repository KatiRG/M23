Interactive web visualization of the support network of the M23 rebel group in DRC based on UN report S/2012/843. The visualzation was published in the following commentary:

Nangini, C, Jas, M, Fernandes, H.L. and Muggah, R 2014. Visualizing Armed Groups: The Democratic Republic of the Congo’s M23 in Focus. Stability: International Journal of Security and Development 3(1):5, DOI: http://dx.doi.org/10.5334/sta.dd

The visualization was coded using the hierarchical edge bundling algorithm in d3js.
